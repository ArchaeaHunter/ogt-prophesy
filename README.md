# OGT Prophesy - An experiment to predict OGT from amino-acids frequencies on Archaea

## Repository Overview

The repository contains a notebook with experiments to predict OGT (Optimal Growth Temperature) with the less biases as possible, and an ad-hoc script with utilities to test easily, fast and by reproductive way differents machines learning strategies to predict OGT from feature vectors of any kind representing the organism genome. 

To make a deeper discovery, let's see the notebook as explained above. It contains all meaningful explanations about the problem.

**Prerequisities :** Python 3.13+

```bash
git clone git@github.com:ArchaeaHunter/ogt-prophesy.git
cd ogt-prophesy

uv run jupyter lab notebook_archaea_ogt_prediction.ipynb
```
